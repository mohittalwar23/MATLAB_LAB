## About

This repository contains MATLAB code developed for ECE307, covering fundamental Digital Signal Processing concepts through hands-on lab experiments. The implementations include signal generation, discrete transforms, convolution, and properties verification — building practical intuition alongside the theoretical coursework.

---

## Prerequisites

- MATLAB (any recent version) installed on your machine
- Basic familiarity with discrete-time signals and systems

---

## Experiments

| # | Title |
|---|---|
| Experiment 1 | Plotting basic discrete signals using a custom function and calculating their energies |
| Experiment 2 | Discrete convolution using a custom function and algorithm |
| Experiment 3 | Discrete Time Fourier Transform and sampling |
| Experiment 4 | Discrete Time Fourier Transform (continued) |
| Experiment 5 | Inverse Discrete Time Fourier Transform |
| Experiment 6 | Verification of DTFT properties |

Clone the repository and open any experiment folder in MATLAB. Each experiment is self-contained and can be run independently.
